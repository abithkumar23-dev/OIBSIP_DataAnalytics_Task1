# Oasis Infobyte Internship

## Task 1: Exploratory Data Analysis (EDA) on Retail Sales Data

This project performs exploratory data analysis on a retail sales dataset.

### Tools Used
- Python
- Pandas
- Matplotlib
- Seaborn

### Analysis Performed
- Data overview
- Missing value check
- Sales distribution visualization
- Product category analysis
- Age vs purchase amount visualization

This project was completed as part of the Oasis Infobyte Data Analytics Internship.
